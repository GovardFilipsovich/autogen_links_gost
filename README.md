# autogen_links_gost

## Назначение
Данный скрипт имеет довольно узкое назначение: оформление списка ссылок по госту для учебных отчетов.

При помощи beautifulsoup скрипт берет заголовки для ссылок и берет текущее время для даты обращения.

Но список должен содержать также ссылки на книги, что не позволяет дооформить данные ссылки в виде списка.

## Применение

1) Записываем список ссылок в файл links
2) прописываем:
```bash
python3 script
```
3) Копируем вывод и вставляем в ворд. Далее форматируем как список.
